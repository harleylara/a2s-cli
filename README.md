# `a2s-cli`

## Install

Intall dependencies:

for Ubuntu systems:
```
sudo apt update
sudo apt install -y software-properties-common
sudo add-apt-repository -y ppa:apptainer/ppa
sudo apt update
sudo apt install -y apptainer
```

Install tool:

```
git clone https://github.com/harleylara/a2s-cli.git
cd a2s-cli
pip install .
```
